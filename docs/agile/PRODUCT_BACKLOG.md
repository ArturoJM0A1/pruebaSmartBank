# Product Backlog - SmartBank

## Epic: SmartBank Digital Banking

A comprehensive digital banking application that allows users to manage accounts, make transfers, pay bills, and monitor their financial activity.

---

## Backlog Management

**Backlog Grooming**: Weekly session to refine estimates, add acceptance criteria, and reprioritize.

**Prioritization Method**: MoSCoW (Must, Should, Could, Won't)

**Story Points**: Fibonacci scale (1, 2, 3, 5, 8, 13, 21)

## Story Writing Guidelines

### INVEST Criteria
Each story must be:
- **I**ndependent: Can be developed without other stories
- **N**egotiable: Details can be discussed
- **V**aluable: Delivers value to users
- **E**stimable: Team can estimate effort
- **S**mall: Can be completed in one sprint
- **T**estable: Has clear acceptance criteria

### Story Splitting
- Split by workflow steps (login → dashboard → accounts)
- Split by data type (accounts → cards → transactions)
- Split by CRUD operations (view → create → update → delete)
- Split by platform (mobile → desktop)
